    2. 使用SDK¶ ↑

    通过Client生成客户端实例，API接口说明参见“mos/sdk/v1/client.rb文档或doc/Client.html”。
    示例代码如下:

    require "mos/sdk/v1/client"  
key = 'MOS_ACCESS_KEY'  
secret = 'MOS_ACCESS_SECRET'  
url = 'https://mosapi.meituan.com/mcsapi/v1'  
cli = Client(key, secret, url)  
balance = cli.GetBalance()  
puts balance
