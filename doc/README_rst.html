    Ruby SDK

    ¶ ↑

    为了方便客户使用美团云服务，我们提供了方便调用MOS API的Ruby SDK，用户可以通过调用Ruby SDK在自己的程序中实现对美团云资源的控制。

    安装

    使用gem安装 ~~~~~~~~~~~

    在命令行下可以直接使用gem进行安装

    sudo gem install mos-sdk

    从源码安装 ~~~~~~~~~~

    从 `github下载最新版 <github.com/meituan/mcsapi_ruby.git

    进入mcsapi_ruby目录执行下述命令安装ruby SDK

    sudo gem install

    访问MOS控制台的 `API页面 <`_“>mcsapi-python.readthedocs.org/zh_CN/latest/client.html>`_ 。

    示例代码如下：

    require “mos-sdk”

    key = 'MOS_ACCESS_KEY' secret = 'MOS_ACCESS_SECRET' url = 'MOS_ACCESS_URL'

    cli = MOS::Client(key, secret, url)

    balance = cli.get_balance() puts balance
